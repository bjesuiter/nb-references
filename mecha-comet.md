## Mecha Comet

### Summary
A modular Linux handheld computer that's truly yours to own, build, and mod. Pocket-sized powerhouse for geeks by geeks.

### Specs

| Component | Details |
|-----------|---------|
| CPU | i.MX8M Plus (4-core A53) or i.MX 95 (6-core A55) |
| GPU | Integrated + NPU |
| RAM | 2GB, 4GB, or 8GB LPDDR5 |
| Storage | 64GB or 128GB eMMC, expandable via NVMe SSD |
| Display | 3.92" AMOLED (550 nits), capacitive touch |
| Battery | 4100 mAh |
| OS | Mechanix OS (Linux 6.12) |
| Security | CC EAL 6+ Trust Anchor |
| Size | 155 x 73 x 14 mm, 225g |

### Connectivity
- WiFi 802.11ac, BT 5.4
- Physical SIM card (LTE/5G)
- 1x HDMI, 2x USB-C
- 40 magnetic extension pins

### Extension Modules
- **LTE/5G Modem** — Mobile data, calling, SMS
- **NVMe SSD** — Up to 2TB storage (400-600 MB/s)
- **LoRa/Meshtastic** — Decentralized p2p communication
- **AI Accelerator** — Up to 13 TOPS for computer vision
- **FPGA Support** — Custom hardware acceleration

### Open Source
- Bootloader, kernel, and rootfs are open-source
- Open-hardware extensions
- GitHub: github.com/mecha-org

### Use Cases
- Pocket development machine
- Mobile Linux tinkering
- IoT/robotics controller
- Privacy-focused mobile computing
- Learning Linux on the go
