## Reference

### Summary
A Rust project template that uses Zig as a C compiler wrapper to cross-compile binaries for macOS, Linux, and Windows without requiring Docker.

### Content
This template demonstrates how to use Zig's CC (C compiler) capabilities to cross-compile Rust projects across multiple operating systems, simplifying CI/CD pipelines for multi-platform releases.
