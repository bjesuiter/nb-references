## Summary

**WXT** is a next-gen web extension framework that makes web extension development faster than ever. It unifies Chrome and Firefox development with excellent developer experience.

## Features

- **Universal Browser Support:** Works with all browsers (Chrome, Firefox, Safari, Edge, etc.)
- **Manifest Support:** Supports both MV2 and MV3
- **Lightning Fast Dev Mode:** HMR for UI, fast reload for content/background scripts
- **TypeScript:** Type-safe development by default
- **File-based Entrypoints:** Automatic discovery of content/background/popup scripts
- **Auto-imports:** Automatic imports for utils and helpers
- **Frontend Framework Agnostic:** Works with Vue, React, Svelte, etc.
- **Automated Publishing:** Auto-zip, upload, submit, and publish extensions
- **Bundle Analysis:** Tools for analyzing and minimizing extension size
- **Module System:** Reusable code between extensions
- **Remote Imports:** Download and bundle remote URL imports

## Quick Start

```bash
# npm
npx wxt@latest init

# pnpm
pnpm dlx wxt@latest init

# bun
bunx wxt@latest init
```
