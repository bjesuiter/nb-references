## Overview

Mona Sans and Hubot Sans are two open-source variable fonts from GitHub, designed with Degarism and inspired by industrial-era grotesques.

## Mona Sans

**A strong and versatile typeface** designed for use across:
- Product interfaces
- Web applications
- Print materials

### Variable Axes
- **Weight**: Ultra thin → Extra heavy
- **Width**: Condensed → Expanded
- **Slant**: Regular → Italics

### Characteristics
- Industrial-era grotesque inspiration
- Highly readable at all sizes
- Works well for both body text and headings

## Hubot Sans

**Mona's robotic sidekick** — more geometric accents with a technical feel:
- Perfect for headers and pull-quotes
- Unique technical aesthetic
- Complements Mona Sans

### Variable Axes
Same flexible axes as Mona Sans:
- Weight, Width, Slant

## Usage

- **Download**: https://github.com/mona-sans/releases/latest
- **License**: OFL (Open Font License) — free to use on sites and projects
- **Variable Fonts**: Package multiple axes into one file for granular control

## JB's Notes

Recommended for:

- Web applications needing a versatile typeface
- Print materials (Mona Sans)
- Technical documentation and headers (Hubot Sans)
