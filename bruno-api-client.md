## Reference

### Summary
Bruno is a modern, open-source API client designed to replace Postman/Insomnia. Key differentiators:

- **Offline-first**: No cloud sync, data stays on your device
- **Git-friendly**: Stores collections as plain text using "Bru" markup language
- **Lightweight**: Much lighter than Postman's electron app
- **Cross-platform**: Mac, Windows, Linux

### Key Features
- Request collection management
- Environment variables
- Scripting with Bru language
- Git version control integration
- Import from Postman/Insomnia

### Installation
```bash
# macOS
brew install bruno

# Windows
winget install Bruno.Bruno

# Linux (AUR)
yay -S bruno

# Linux (Flatpak)
flatpak install com.usebruno.Bruno
```

### Why Bruno
- Your API collections become code-reviewable
- Works offline 100%
- No account required
- MIT licensed
- Active development community
