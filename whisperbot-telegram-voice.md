## Summary

A Telegram bot that transcribes voice messages locally using OpenAI Whisper. Useful reference for implementing voice message transcription in the mobile coding app's PWA or any Telegram-integrated features.

## Key Features

- Runs locally (privacy-preserving)
- Uses OpenAI Whisper for transcription
- Handles Telegram voice messages
- No external API calls for transcription

## Use Cases

- Voice notes in coding sessions
- Mobile-friendly input methods
- Accessibility features
