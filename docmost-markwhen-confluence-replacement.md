## Docmost

**Enterprise-ready Wiki for Teams** - Confluence replacement with on-premises deployment.

### Key Features
- **Real-time Collaboration**: Live cursors, instant sync
- **RBAC Permissions**: Groups, granular permissions
- **Team Spaces**: Dedicated spaces for teams/projects/departments
- **Diagrams**: Built-in Draw.io, Excalidraw, Mermaid support
- **Comments**: Inline commenting on pages

### Air-gapped / On-Premises
- Deploy on your own servers
- Isolated environments
- Private data centers
- Complete data control

### Compliance
- ITAR ready
- GDPR data sovereignty
- FedRAMP

### Enterprise Auth
- SAML/SSO integration
- Multi-factor authentication

### Migration
Import from Confluence, Notion, HTML, Markdown files.

## Markwhen

**Markdown language for Gantt charts** - Timeline and project planning.

- Write Gantt charts in markdown
- Visual timeline output
- Lightweight, text-based approach

---

## Use Together

- **Docmost** for team documentation, knowledge base
- **Markwhen** for project planning and timelines
- Both support markdown, both can be deployed on-premises
- Good stack for airgapped or compliance-focused environments
