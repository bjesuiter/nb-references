## Summary

A component library for creative WebGPU effects in the browser. Features 70+ production-ready components including WaveDistortion, LinearGradient, Swirl, and more.

## Key Features

- **WebGPU-native:** Modern GPU-accelerated shader effects
- **70+ components:** Pre-built effects (gradients, distortions, blurs)
- **Stackable:** Components can be combined, nested, masked, and blended
- **Visual Editor:** Design effects in GUI, export production-ready code
- **Framework agnostic:** Works with any modern frontend

## Use Cases

- Interactive visual effects for landing pages
- Creative UI components with shader animations
- Design-to-code workflows for web graphics
