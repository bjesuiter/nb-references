## Reference

### Summary
Yaak is a modern, fast, and secure API client created by **Greg (the creator of Insomnia)**. Built from scratch after selling Insomnia, addressing bloat and privacy concerns.

### Key Features
- **Local-only**: No telemetry, data stays on your device
- **Git-friendly**: Store as plain-text for version control
- **Encrypted secrets**: Keychain integration for sensitive data
- **Fast**: Lightweight, responsive UI
- **Node.js plugins**: Extensible authentication, template functions
- **Multi-protocol**: REST + GraphQL in one workspace
- **CURL conversion**: Paste Curl → auto-format to request

### Features at a Glance
- Request chaining & variables
- Auth inheritance
- Rich previews (JSON, images, etc.)
- Command palette
- Batch send requests
- Custom proxy support
- Postman/Insomnia import

### Installation
- **macOS**: Download from website or Homebrew
- **Windows**: Download installer
- **Linux**: Download .deb or AppImage
