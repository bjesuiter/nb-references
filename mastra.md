## Reference

### Summary
Mastra is an AI agent framework from the team behind Gatsby. Built for TypeScript with everything needed to go from prototype to production.

### Key Features

- **Model Routing** - Connect to 40+ providers through one standard interface (OpenAI, Anthropic, Gemini, etc.)

- **Agents** - Build autonomous agents that use LLMs and tools to solve open-ended tasks

- **Workflows** - Graph-based workflow engine with intuitive syntax:
  - `.then()` - Chain commands
  - `.branch()` - Conditional branches
  - `.parallel()` - Parallel execution

- **Human-in-the-loop** - Suspend agent/workflow and await user input/approval before resuming

- **Memory** - Conversation history, working memory, semantic recall

- **MCP Servers** - Author Model Context Protocol servers

- **Integrations** - Works with React, Next.js, Node.js, Vercel AI SDK, CopilotKit

- **Production Essentials** - Built-in evals and observability

### Quick Start
```bash
npm create mastra@latest
```

### Why It Matters
- TypeScript-first AI framework
- From Gatsby team (established JS ecosystem)
- Easy to prototype, reliable in production
- Chainable workflow syntax is intuitive
