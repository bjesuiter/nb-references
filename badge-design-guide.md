## Reference

Haz (@haz.dev) shared a visual guide on styling UI badges better.

### Key Techniques

1. Start with a solid background badge (e.g., green "Success")
2. Add a blended dark background
3. Highlight text with color
4. Add an icon (checkmark)
5. Use `margin-inline-start: -4px` (negative margin) for better icon-text alignment
