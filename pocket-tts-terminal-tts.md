## Quick Start

```bash
# Install via Homebrew (if available)
brew install pocket-tts

# Or via npm
npm install -g pocket-tts

# Basic usage
pocket-tts "Hello, world!"

# With specific voice
pocket-tts --voice "Victoria" "Welcome back"
```
