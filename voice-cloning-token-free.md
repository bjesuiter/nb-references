## Options

### Open Source / Local Models
- **Coqui TTS** - Open-source TTS with voice cloning capabilities
- **Bark** - Voice cloning with minimal training data
- **XTTS-v2** (Coqui) - Multi-lingual voice cloning
- **Microsoft SpeechT5** - HuggingFace based
- **OpenAI's Whisper + replacement TTS** - Local pipeline

### Resources
- HuggingFace has various voice cloning models
- Run locally with GPU acceleration

### Notes
- Requires some setup but no per-token costs
- Quality varies by model and training data
