### Techniques

**Classical Rhetoric (Aristotle's 3 Appeals)**
- **Ethos** — Credibility and character
- **Pathos** — Emotional appeal
- **Logos** — Logical argument

**Modern Speaking Framework**
- Start with hook/attention
- State main point clearly
- Support with evidence/stories
- End with call-to-action or memorable close

**Building Vocabulary**
- Read widely (fiction, non-fiction, journalism)
- Learn one new word daily
- Practice using new words in conversation
- Read aloud to practice articulation

### Quick Tips

- **Pause more** — Silence = confidence
- **Avoid filler words** — "Um", "like", "you know"
- **Structure first** — Think before speaking
- **Listen actively** — Good speakers are good listeners
- **Read fiction** — Improves empathy and vocabulary
- **Record yourself** — Identify areas to improve
