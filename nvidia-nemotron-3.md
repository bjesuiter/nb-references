## Summary

NVIDIA Nemotron 3 is a family of open models for agentic AI applications. Released December 2025.

## Model Family

| Model | Description |
|-------|-------------|
| **Nemotron 3 Nano** | Smallest, efficient for edge/local deployment |
| **Nemotron 3 Super** | Coming H1 2026 |
| **Nemotron 3 Ultra** | Coming H1 2026 |

## Key Features

- **Hybrid Transformer-Mamba MoE architecture** (Nano)

- **Advanced reasoning and agentic capabilities**
- **High efficiency** for local deployment
- **NVFP4 4-bit precision** training support (Super/Ultra)
- **Multi-token prediction** for inference throughput (Super/Ultra)
- **Latent MoE** for expert specialization (Super/Ultra)

## Open Model

- NVIDIA releases training data
- Tools for customization and fine-tuning
- Good for building AI agents that take actions on computers or web

## Future (H1 2026)

- Nemotron 3 Super and Ultra will incorporate:
  - Latent MoE for greater expert specialization
  - Multi-token prediction for boosted inference throughput
  - NVFP4 4-bit precision for cost-accuracy
