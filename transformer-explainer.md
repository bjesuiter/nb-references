## Summary

Transformer Explainer is an interactive web-based tool that visually explains how transformers work, using the GPT-2 (124M parameter) model as an example. It allows users to see self-attention mechanisms in action and understand the architectural components shared with modern LLMs.

## Key Features

- Visual representation of transformer components
- Interactive attention mechanism demonstration
- Based on real GPT-2 architecture
- No installation required - runs in browser

## Why This Resource

Perfect for building intuition about transformers before diving into code or mathematical details.
