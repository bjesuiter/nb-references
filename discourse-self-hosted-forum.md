## Reference

**Source:** https://www.discourse.org/

### Summary

Discourse is a modern, open-source forum platform designed for community discussions. It's fully self-hostable and known for its beautiful design and rich feature set.

### Key Features

- Modern, responsive UI
- Built with Ruby on Rails and Ember.js
- Rich markdown support
- Gamification (trust levels, badges)
- Powerful moderation tools
- Single sign-on (SSO) integration
- Docker-based deployment
- Plugin system for extensibility

### Use Cases

- Community forums
- Support forums
- Developer discussions
- Announcements and announcements

### Deployment

- Official Docker-based deployment
- Can be self-hosted on any server
- Cloud hosting available at discourse.org
