## Reference

PDSLS is a web-based explorer for the AT Protocol (atproto) that allows browsing public data on Personal Data Servers (PDSes). It's a mirror of the original Bluesky/ATProto data exploration tool.

### What it does
- Browse public data on various PDSes
- Explore the decentralized AT Protocol data structure
- View records, repositories, and collections across different servers
