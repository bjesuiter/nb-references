## Summary

A bookmark and content sync service that saves links from various sources as clean markdown. Similar in concept to LoreVault idea.

## Features

- **X/Twitter bookmarks** - Sync saved bookmarks automatically, stores tweet text + links as markdown
- **Article feeds** - Follow X article feeds in background, save long-form posts as markdown
- **YouTube content** - Save transcripts and key comments, follow channels automatically
- **RSS feeds** - Automatically save new posts from blogs/news as they're published
- **Email newsletters** - Personal inbox for newsletters, save each issue automatically
- **Chrome extension** - Save any tab with one click, content extracted as clean markdown
- **Mobile** - iOS/Android share sheet integration
- **Web app** - Browse and search saved markdown in unified feed
- **API & CLI** - Full REST API and command-line tool for programmatic access
- **Agent skill** - Teach AI agents to search, read, and use bookmarks as context

## Comparison to LoreVault Idea

Keep.md is a similar concept - saving bookmarks as markdown with AI agent access. Could be a competitive reference or inspiration for LoreVault features.
