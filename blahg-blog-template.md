## Overview

**Blahg** — A simple, elegant blog template built with Astro and TinaCMS by [Cassidy](https://cassidoo.co).

## Features

### Core
- **Astro** — Static site generator
- **TinaCMS** — Git-based headless CMS
- **Markdown posts** — Simple `.md` files
- **Dynamic tags** — Auto-generated from post tags

### Content
- Posts as simple markdown files
- Tag-based organization
- RSS feed: https://blahg.netlify.app/rss.xml

### Design
- Clean, minimal aesthetic
- Homepage with recent posts
- Tag archive pages

## Quick Start

```bash
# Clone the template
git clone https://github.com/cassidoo/blahg.git
cd blahg

# Install dependencies
npm install

# Start development server
npm run dev
```
