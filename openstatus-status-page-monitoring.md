## My Reference

### Summary
Open-source status page and uptime monitoring solution. Communicates openly during incidents, shares real-time updates, and automatically populates status from uptime monitoring.

### Features

**Status Pages**
- Custom branding, themes, custom domains
- Public or password-protected pages
- Status reports & maintenance windows
- Subscriptions: email, RSS/Atom, SSH

**Monitoring**
- 28 regions across multi-cloud (Fly.io, Koyeb, Railway)
- API monitoring: assertions, thresholds, headers, body validation
- Monitoring as Code: YAML config, GitHub Actions, CLI
- Private locations: 8.5MB Docker image for internal monitoring
- OpenTelemetry export

**Integrations**
- Slack, Discord, email notifications
- OpenTelemetry metrics export

### Pricing

**Free Tier:**
- 1 monitor
- 1 status page (3 components)
- Cron jobs (min 10m)
- No credit card required

**Paid:** Available for scaling
