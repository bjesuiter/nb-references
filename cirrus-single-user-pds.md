## My Reference

### Summary
Cirrus is the easiest single-user ATProto PDS, running on Cloudflare Workers with Durable Objects and R2.

### Key Features
- **Single-user PDS** - Designed specifically for hosting one account
- **Cloudflare Workers** - Serverless deployment
- **Durable Objects + R2** - Storage backend
- **Easy CLI** - Simple setup and migration from other PDSs
- **Minimal feature set** - Focus on core ATProto functionality
- **Lightweight** - Designed to be as simple as possible

### Use Cases
- Self-host your own PDS for full data ownership
- Migrate from official Bluesky PDS to self-hosted
- Run a personal data server with minimal infrastructure
