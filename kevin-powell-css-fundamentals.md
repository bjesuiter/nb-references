## Summary

Comprehensive CSS fundamentals course by Kevin Powell on Frontend Masters. Covers essential CSS concepts from basics to advanced responsive design patterns.

## Course Details

- **Instructor:** Kevin Powell
- **Platform:** Frontend Masters
- **Published:** February 6, 2026
- **Duration:** ~10 hours

## Course Outline

### CSS Essentials (1h 11m)
- Applying CSS methods & external stylesheets
- Selectors: class, ID, compound, descendant
- Modern CSS nesting
- Dev Tools debugging
- Specificity & cascade
- Color formats
- Inheritance
- Pseudo-classes (hover, focus, focus-visible)

### Typography (57m)
- Text styling & alignment
- Font sizing & units
- Font families & stacks
- @font-face & local fonts
- CSS custom properties (variables)

### Box Model (59m)
- Content, padding, border, margin
- Logical properties
- Margin collapsing
- box-sizing: border-box

### Backgrounds & Images (34m)
- background-image & gradients
- Linear, radial, conic gradients
- Image sizing & constraints
- CSS resets

### CSS Grid Layout (1h 3m)
- display: grid basics
- grid-template-columns & fr units
- Responsive grids with media queries
- Spanning rows/columns
- Alignment & justification
- Grid Areas for responsive layouts

### Flexbox Layout (55m)
- Flexbox vs Grid (1D vs 2D)
- align-items & justify-content
- Navigation bars
- flex-grow & flex-shrink
- Flexbox patterns

### Responsive Layouts (1h 20m)
- Fluid grids & flexible images
- Media queries & breakpoints
- Intrinsic design patterns
- auto-fit & auto-fill
- Responsive typography (clamp, Utopia)
- Container queries

### Positioning (1h 5m)
- static, fixed, sticky, relative, absolute
- z-index & stacking contexts

### Decorations & Effects (57m)
- Box shadows & border radius
- aspect-ratio & object-fit
- Pseudo-elements (::before, ::after)

### Animation & Interaction (1h 18m)
- Form feedback & validation
- :has() selector
- CSS transitions & animations
- CSS transforms (scale, rotate, translate)
- Performance considerations

## Key Takeaways

- Confidence to work with vanilla CSS or utility-first approaches like Tailwind
- Understanding of when to use Grid vs Flexbox
- Modern responsive patterns including container queries
- Accessible form styling & user feedback
- Production-ready animation techniques
