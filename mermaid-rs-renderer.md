## Reference

### Summary
A Rust-based Mermaid diagram renderer that allows server-side rendering without needing a browser for SVG generation.

### Content
This project provides a Rust implementation for rendering Mermaid diagrams, useful for generating diagrams programmatically without relying on JavaScript browser rendering.
