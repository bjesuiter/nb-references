# Raspberry Pi App Store Options

## Recommended: Pi-Apps
- GitHub: https://github.com/Botspot/pi-apps
- Web: https://pi-apps.io
- GUI app store for Raspberry Pi
- Large collection of pre-compiled apps
- Easy installation: `wget -qO- https://raw.githubusercontent.com/Botspot/pi-apps/master/install | bash`

## Alternatives
- **Homebrew**: https://docs.brew.sh/Installation.html#linux
- **apt**: Standard package manager
- **Flatpak**: `sudo apt install flatpak`, then use GNOME Software
