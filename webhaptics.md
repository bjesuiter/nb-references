## Reference

### Summary
An npm library that provides haptic feedback functionality for mobile web applications. Allows developers to trigger vibration/haptic feedback on mobile devices via JavaScript API.

### Content
**WebHaptics** - Haptic feedback for the mobile web.

Likely provides a simple API like:
- `haptics.impact(style)` - Impact feedback
- `haptics.notification(type)` - Notification feedback  
- `haptics.selection()` - Selection feedback

Useful for:
- Mobile web apps needing tactile feedback
- PWA improvements
- Better mobile UX
