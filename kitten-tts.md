## Kitten TTS

### Summary
An ultra-lightweight local TTS model with only 25MB model size, designed for resource-constrained environments while maintaining reasonable speech quality.

### Key Features
- **Size:** 25MB (extra small)
- **Local execution:** No API calls needed
- **Open source:** Available on GitHub
- **Lightweight:** Suitable for edge devices, containers, or low-resource machines

### Use Cases
- IoT devices with limited storage
- Docker containers needing TTS
- Battery-powered devices
- Fallback TTS when API access is unavailable
- Privacy-sensitive applications (offline processing)
