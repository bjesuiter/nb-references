## My Reference

### Summary

Summarize.sh is a tool for summarizing links and content using AI. It can take any URL and generate a concise summary of its contents.

**Usage:**
```bash
curl -s https://summarize.sh/ | bash -s <url>
# or
curl -s https://summarize.sh/<url>
```

### Content

Useful for:
- Summarizing articles before storing as references
- Quick content overview of long pages
- Reducing context size for stored references
- Creating summaries for note-taking workflows

**Integration with Clawdbot reference storage:**
- Can use `summarize.sh` to generate summaries before storing references
- Reduces storage size and improves searchability
- Preserves key information in compact form
