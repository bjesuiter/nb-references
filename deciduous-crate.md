## My Reference

### Summary

Deciduous is a Rust crate that provides decision graph functionality for code repositories. Useful for AI coding assistants to follow structured decision paths.

### Content

Useful for:
- Implementing decision trees/graphs in Rust projects
- Providing AI agents with structured decision-making paths
- Creating explicit logic flows for complex coding decisions

**Installation:**
```bash
cargo add deciduous
```
