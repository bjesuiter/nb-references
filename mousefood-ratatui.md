## Reference

### Summary
A Rust backend implementation for Ratatui that targets microcontroller displays, enabling terminal UI rendering on embedded hardware.

### Content
Mousefood provides the bridge between Ratatui's TUI rendering capabilities and microcontroller display hardware, allowing TUI applications to run on embedded systems.
