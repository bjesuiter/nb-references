## My Reference

### Summary

MarkWhen is a markdown-like language specifically designed for expressing things that happen over time. It supports logs, gantt charts, blogs, feeds, notes, journals, diaries, todos, timelines, and calendars.

**Key strengths:**
- Plain text syntax (markdown-like)
- Text-to-diagram workflows
- Timing diagrams
- Gantt charts
- Timeline visualizations

### Content

Useful for:
- Project planning with plain text
- Creating timing diagrams from markup
- Journaling with automatic visualization
- Gantt charts in markdown documents
- Sequence/timeline diagrams

**Great for AI-assisted diagram generation** - simple text input, automatic diagram output.
