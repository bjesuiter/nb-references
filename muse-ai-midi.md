## Summary

**Muse** is an AI-native MIDI composer and editor. It generates fully editable MIDI that can be exported to any DAW, serving as an AI co-writer for music production.

## Features

- **MIDI Generation:** Create multi-track MIDI with AI
- **Editable Output:** All notes editable in piano roll interface
- **Chat Refinement:** Refine generated music through conversation
- **DAW Integration:** Export to any DAW (Ableton, Logic, FL Studio, etc.)
- **Musical Understanding:** Composers with harmonic function, voice leading, and music theory
- **Collaboration:** Create full arrangements, extend melodies, add accompaniment

## Tools / Ecosystem

Three deployment options:
1. **Muse Desktop** - Native desktop app
2. **Muse Web** - Browser-based interface
3. **Muse DAW** - Plugin for direct DAW integration

## Use Cases

- Break through writer's block with AI-generated ideas
- Generate melodies, bass lines, chord progressions
- Create full arrangements from a seed melody
- Rapid prototyping for music production
- Learn music theory through AI collaboration

## Why It Matters

MIDI is the universal language of music production. AI that generates editable MIDI allows producers to:
- Keep full creative control
- Use AI as collaborator, not replacement
- Integrate seamlessly into existing workflows
- Export directly to production tools
