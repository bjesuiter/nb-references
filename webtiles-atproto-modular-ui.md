## Summary

WebTiles are **composable web docs and apps** that can be safely used in arbitrary contexts and assembled from multiple sources. Built on the **DASL** specification with content addressing, making them ideal for AT Protocol, social media, chat, agents, and malleable software.

## Key Features

### Security & Privacy
- Sandboxed execution in arbitrary contexts
- Safe composition from multiple sources
- Content-addressed for verification

### Modularity
- Small, self-contained UI units
- Mix and match from any source
- Build complex UIs from simple blocks

### AT Protocol Integration
- Published directly to AT Protocol
- Discoverable across the network
- Portable across applications
