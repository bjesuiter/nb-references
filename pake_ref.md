## Reference

### Summary
Turn any webpage into a desktop app with one command. Built with Rust/Tauri, ~20x smaller than Electron (around 5MB).

### Features
- Lightweight: ~5MB packages (vs Electron's 100MB+)
- Fast: Rust-based, lower memory usage
- Easy: One-command CLI or online building
- Rich: Shortcuts, immersive windows, drag & drop, style customization, ad removal
- Cross-platform: macOS, Windows, Linux

### Usage
- Beginners: Download ready-made packages or use GitHub Actions online build
- Developers: Install CLI tool for one-command packaging with custom icons/window settings
