## My Reference

### Summary

Amore is a GUI tool that simplifies distributing macOS apps outside the App Store with over-the-air (OTA) updates using Sparkle. Created by lucas.love, used by projects like whyFi.

### Features

- **Easy Sparkle Setup** - Drag-and-drop app integration in <5 minutes
- **DMG Creation** - Auto-packages apps with custom backgrounds
- **S3 Compatible Storage** - Use your own storage or Amore's servers
- **Beta Channels** - Test updates before public release
- **Phased Rollouts** - Gradual rollout over 7 days
- **Critical Updates** - Mandatory updates that can't be skipped
- **Signing & Key Management** - Secure private key handling, never leaves your computer
- **Custom Domains** - Independent update fetching, no vendor lock-in

### Technical Details

- Built on top of Sparkle (the de facto standard for macOS OTA updates)
- Fully compatible with existing Sparkle appcasts
- Automatic migration from existing Sparkle setups
- Notarizing support for Apple Gatekeeper

### Use Cases

- Distribute macOS apps outside App Store
- Ship updates to users without requiring App Store review
- Manage beta testing programs
- Maintain independent distribution channels
