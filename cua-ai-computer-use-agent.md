## Overview

Cua.ai is a Computer Use Agent (CUA) platform that enables AI agents to see screens, click buttons, type text, and run code on desktop interfaces.

## Key Features

### Computer Use Stack
- **Unified interface** for computer use models
- **Structured outputs** and multi-turn conversations
- **Custom tools** integration

### Platform Support
- macOS, Windows, Linux
- Browser and phone automation
- MCP (Model Context Protocol) support

### Cloud Sandboxes
- **One-line deploy** from terminal
- **Unlimited scale** without infrastructure management
- **Cross-platform** environments (Linux, Windows, macOS)
- **Pay-as-you-go** billing

### Data & Training
- **UI dataset generation** with bounding boxes and metadata
- **Trajectory recording** for agent interactions
- **Benchmark evals** for safety and accuracy
- **HuggingFace export** in Arrow/Parquet format

## Open Source Components

- Agent SDK
- Computer SDK
- macOS virtualization
- Docker images

## Use Cases

- Desktop application automation
- Web testing and scraping
- Multi-step workflow automation
- UI/UX testing with AI agents
