## My Reference

### Summary
Blento allows you to create beautiful personal websites with data saved to your Bluesky profile. Easy to use, free, optionally self-hosted, open source.

### Key Features
- **Easy to use & free** - No cost to create your site
- **Data from Bluesky** - Profile data pulled from your Bluesky account
- **Optional self-hosting** - Deploy on your own infrastructure
- **Open source** - https://github.com/flo-bit/blento

### Use Cases
- Personal homepage linking Bluesky profile
- Portfolio site with Bluesky as CMS
- Simple static site powered by ATProto data
