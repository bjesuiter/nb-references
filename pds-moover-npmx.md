## Reference

### Summary
PDS MOOver is a web-based tool for migrating Bluesky accounts from one PDS (Personal Data Server) to another. Specifically supports migration to npmx.social.

**Key features:**
- Web-based migration interface
- Login with old PDS credentials
- Setup new handle/email on npmx.social
- Clear risk disclosure about account migrations
